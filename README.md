# Flixmedia Mobile SDK Hub

This repository is an internal hub for all Flixmedia mobile SDKs and related projects.

It provides quick access to the repositories used to integrate Flixmedia content into mobile applications.

> [!WARNING]
> Internal repository — some linked repositories are private.

---

## Overview

Flixmedia provides SDKs for multiple mobile platforms:

| Platform | Repository |
| --- | --- |
| iOS SDK | [FlixMediaSDK](https://github.com/FlixmediaTech/FlixMediaSDK) |
| iOS Test App | [FlixTestApp](https://github.com/FlixmediaTech/FlixTestApp) |
| Android SDK | [FlixMediaAndroidSDK](https://github.com/FlixmediaTech/FlixMediaAndroidSDK) |
| Android test app | [FlixTestAndroidApp](https://github.com/FlixmediaTech/FlixTestAndroidApp) |
| React Native test app and module package release | [FlixReactTestApp](https://github.com/FlixmediaTech/FlixReactTestApp) |
| React Native Inpage Module | [react-native-flix-inpage](https://github.com/FlixmediaTech/react-native-flix-inpage) |
| Flutter Inpage Plugin | [flix_inpage](https://github.com/FlixmediaTech/flix_inpage) |
| Flutter test app | [flix_test](https://github.com/FlixmediaTech/flix_test) |

## Architecture

Flixmedia mobile SDK ecosystem consists of:

```text
Mobile Apps
├── Native SDKs
│   ├── iOS SDK
│   └── Android SDK
└── Cross-Platform SDKs
    ├── React Native SDK
    └── Flutter SDK
```

Each SDK is maintained in a separate repository to allow independent releases and platform-specific development.
